# chenguandan.github.io

GitHub Pages 用户主页仓库。本身只承担两件事：

1. **`robots.txt`**：告诉爬虫 sitemap 在哪（GitHub Pages 项目仓库 `/blog/` 下的 sitemap 必须由根域名的 `robots.txt` 公开）。
2. **`index.html`**：把访问根域名 `https://chenguandan.github.io/` 的用户重定向到 `/blog/`。

实际内容都在 [`chenguandan/blog`](https://github.com/chenguandan/blog) 仓库。
